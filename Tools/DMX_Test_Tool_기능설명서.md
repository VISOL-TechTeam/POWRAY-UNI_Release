# Visol DMX Test Tool 기능 설명서

## 개요

`Visol DMX Test Tool`은 Windows PC에서 실행하는 DMX512 송신 테스트 프로그램이다.  
USB-DMX 또는 USB-Serial 장치를 통해 POWRAY-UNI 장비의 유선 DMX 수신 기능, 채널 디밍 반응, 디밍 커브, 반복 파형 동작을 확인할 수 있다.

사용자는 `DMX_Test_Tool.exe` 파일만 실행하면 된다. 별도 설치 과정은 없으며, USB-Serial 장치 드라이버는 PC에 미리 설치되어 있어야 한다.

## 실행 방법

1. USB-DMX 또는 USB-Serial 장치를 PC에 연결한다.
2. `DMX_Test_Tool.exe`를 실행한다.
3. Windows 보안 경고가 표시되면 사용 환경에 맞게 실행을 허용한다.
4. 프로그램 화면에서 COM 포트를 선택하고 `Connect`를 누른다.

## DMX 송신 방식

프로그램은 DMX512 프레임을 다음 순서로 반복 송신한다.

1. Break + MAB 생성
2. Start Code 전송
3. CH1 ~ CH512 데이터 전송
4. 설정된 FPS 간격만큼 대기 후 다음 프레임 송신

Break는 보드레이트 전환 방식으로 생성한다.

| 구간 | 설정 | 설명 |
| --- | --- | --- |
| Break/MAB | `57600 bps`에서 `0x00` 1바이트 송신 | DMX Break와 MAB 생성 |
| DMX Data | `250000 bps`, 8 data bits, no parity, 2 stop bits | DMX512 표준 데이터 송신 |

기본 송신 주기는 `43 FPS`이며, 화면에서 변경할 수 있다.

## 화면 구성

### Serial Connection

USB-DMX 또는 USB-Serial 포트를 선택하고 DMX 송신을 시작/중지하는 영역이다.

| 항목 | 설명 |
| --- | --- |
| `Port` | 사용 가능한 COM 포트 목록 |
| `Refresh` | COM 포트 목록 새로고침 |
| `Baudrate` | 연결 시 사용할 초기 보드레이트. 일반적으로 기본값 `250000` 사용 |
| `Connect` | 선택한 포트에 연결하고 DMX 송신 시작 |
| `Disconnect` | DMX 송신 중지 및 포트 연결 해제 |
| 상태 표시 | 연결 상태 또는 오류 메시지 표시 |

연결 중에는 포트와 보드레이트를 변경할 수 없다. 변경이 필요하면 먼저 `Disconnect`를 누른다.

### DMX TX Settings

DMX 프레임 송신 조건과 화면에 표시할 채널 수를 설정한다.

| 항목 | 범위 | 설명 |
| --- | --- | --- |
| `FPS` | 1 ~ 100 | 초당 송신 프레임 수 |
| `StartCode` | `0x00` ~ `0xFF` | DMX Start Code. 일반 디밍 테스트는 `0x00` 사용 |
| `View CH` | 1 ~ 128 | 한 화면에 표시할 채널 수 |

`FPS`와 `StartCode`는 연결 중에도 변경 가능하며, 변경한 값은 다음 송신 프레임부터 반영된다.

### DMX Channels

DMX 채널 값을 직접 조작하는 영역이다.

| 조작 | 설명 |
| --- | --- |
| 세로 슬라이더 | 해당 채널 값을 0 ~ 255 범위로 조절 |
| 숫자 입력 | 해당 채널 값을 직접 입력 |
| `Prev N CH` | 이전 채널 페이지로 이동 |
| `Next N CH` | 다음 채널 페이지로 이동 |

채널 번호는 `CH 1`부터 시작한다. 프로그램은 전체 512채널 데이터를 유지하며, 화면에는 `View CH`에 설정된 개수만 표시한다.

### 전체 채널 제어

| 버튼/입력 | 설명 |
| --- | --- |
| `Clear All (0)` | CH1 ~ CH512 전체 값을 0으로 설정 |
| `Full All` | CH1 ~ CH512 전체 값을 지정 값으로 설정 |
| `Full All` 옆 숫자 입력 | `Full All` 실행 시 적용할 값. 기본값은 255 |

## Dimming Curve Test

선택한 채널 범위에 디밍 커브를 적용해 밝기 반응을 확인하는 기능이다.

| 항목 | 설명 |
| --- | --- |
| `Start CH` | 커브 테스트 시작 채널 |
| `Count` | 적용할 채널 개수 |
| `Curve` | 적용할 커브 종류 |
| `Level` | 입력 레벨 0 ~ 255 |
| `Apply Curve` | 현재 Level과 Curve를 선택 채널 범위에 즉시 적용 |
| `Step` | Sweep 동작 시 레벨 증가/감소 단위 |
| `Interval(ms)` | Sweep 갱신 주기 |
| `Start Sweep` | Level을 0 ~ 255 범위에서 왕복 변화시키며 반복 적용 |

지원 커브는 다음과 같다.

| 커브 | 설명 |
| --- | --- |
| `Linear` | 입력 레벨을 그대로 사용 |
| `Gamma 2.2` | 감마 2.2 적용 |
| `Gamma 1.8` | 감마 1.8 적용 |
| `Square` | 제곱 커브 적용 |
| `Square Root` | 제곱근 커브 적용 |

## Dimming Wave Test

선택한 채널 범위에 반복 파형을 적용해 동적 디밍 반응을 확인하는 기능이다.

| 항목 | 설명 |
| --- | --- |
| `Wave` | 파형 종류 선택 |
| `Min` | 파형 최소 DMX 값 |
| `Max` | 파형 최대 DMX 값 |
| `Period(ms)` | 파형 1주기 시간 |
| `Cycles` | 선택 채널 범위 내 위상 반복 횟수 |
| `Duty(%)` | Pulse 파형에서 High 유지 비율 |
| `Start Wave` | 파형 테스트 시작 |
| `Stop Wave` | 파형 테스트 중지 |

지원 파형은 다음과 같다.

| 파형 | 설명 |
| --- | --- |
| `Sine` | Min ~ Max 범위를 사인파로 변화 |
| `Pulse` | Duty 비율에 따라 Min/Max를 반복 출력 |

Wave 테스트와 Curve Sweep은 동시에 실행되지 않는다. 하나를 시작하면 다른 하나는 자동으로 중지된다.

## 사용 절차 예시

1. POWRAY-UNI 장비의 DMX 입력 라인에 USB-DMX 또는 USB-Serial 장치를 연결한다.
2. PC에서 `DMX_Test_Tool.exe`를 실행한다.
3. `Refresh`를 눌러 COM 포트 목록을 갱신한다.
4. 대상 COM 포트를 선택한다.
5. `Baudrate`는 기본값 `250000`을 사용한다.
6. `StartCode`가 `0x00`인지 확인한다.
7. `Connect`를 눌러 DMX 송신을 시작한다.
8. 장비 DMX 주소에 맞는 채널 값을 슬라이더 또는 숫자 입력으로 변경한다.
9. 장비의 밝기 반응을 확인한다.
10. 테스트 완료 후 `Disconnect`를 누르고 프로그램을 종료한다.

## 테스트 예시

### 단일 채널 디밍 확인

1. 장비 DMX 주소를 확인한다.
2. 해당 채널이 보이도록 `Prev/Next` 버튼으로 페이지를 이동한다.
3. 해당 채널 슬라이더를 0에서 255까지 천천히 올린다.
4. 장비 밝기가 0%에서 100%까지 변하는지 확인한다.

### 전체 채널 반응 확인

1. `Full All` 옆 값을 `255`로 설정한다.
2. `Full All`을 누른다.
3. 장비가 최대 밝기로 반응하는지 확인한다.
4. `Clear All (0)`을 눌러 전체 출력을 0으로 되돌린다.

### 디밍 커브 확인

1. `Start CH`에 시작 채널을 입력한다.
2. `Count`에 테스트할 채널 개수를 입력한다.
3. `Curve`를 선택한다.
4. `Apply Curve` 또는 `Start Sweep`을 눌러 밝기 변화를 확인한다.

### 반복 파형 확인

1. `Start CH`와 `Count`를 설정한다.
2. `Wave`에서 `Sine` 또는 `Pulse`를 선택한다.
3. `Min`, `Max`, `Period(ms)` 값을 설정한다.
4. `Start Wave`를 눌러 반복 디밍 반응을 확인한다.

## 주의 사항

- DMX512 표준은 `250000 bps`, `8N2` 구성을 사용한다.
- USB-Serial 장치가 `250000 bps` 설정과 빠른 보드레이트 전환을 지원해야 한다.
- 일부 USB-Serial 칩에서는 Break 타이밍 특성이 다를 수 있다.
- 일반 디밍 테스트에서는 `StartCode`를 `0x00`으로 설정해야 한다.
- 장비 DMX 주소와 프로그램 채널 번호가 일치해야 한다.
- 연결 중 포트 목록과 보드레이트를 변경하려면 먼저 `Disconnect`해야 한다.
- 프로그램 종료 시 연결된 포트는 자동으로 닫힌다.

## 문제 해결

| 증상 | 확인 사항 |
| --- | --- |
| COM 포트가 보이지 않음 | USB 연결 상태, 장치 드라이버 설치 상태 확인 후 `Refresh` 실행 |
| 연결 실패 | 다른 프로그램이 같은 COM 포트를 사용 중인지 확인 |
| 장비가 반응하지 않음 | DMX 라인 결선, GND 연결, RS485 방향, Start Code `0x00`, 장비 DMX 주소 확인 |
| `Full All`에는 반응하지만 특정 채널에는 반응하지 않음 | 장비 DMX 주소와 선택한 채널 번호 확인 |
| 일부 채널만 반응 | `View CH`, 페이지 위치, 장비 채널 매핑 확인 |
| 밝기 변화가 부자연스러움 | `Curve`, `Min/Max`, `FPS`, 장비 내부 디밍 보정 설정 확인 |
| 송신 중 오류 발생 | USB 연결 안정성, USB-Serial 칩의 `250000 bps` 지원 여부 확인 |
